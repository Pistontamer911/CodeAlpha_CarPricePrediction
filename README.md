# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict the selling price of used cars using Machine Learning techniques. Various car-related features such as present price, fuel type, transmission type, ownership history, and vehicle age are used to build a regression model that estimates car prices accurately.

## 🎯 Objectives

* Analyze car price data and identify important features.
* Perform data preprocessing and feature engineering.
* Train a machine learning regression model.
* Evaluate model performance using standard metrics.
* Predict car selling prices based on vehicle characteristics.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📂 Dataset Features

* Car Name
* Year
* Selling Price
* Present Price
* Driven Kilometers
* Fuel Type
* Selling Type
* Transmission
* Owner

## 🔄 Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Visualization
6. Model Training
7. Price Prediction
8. Model Evaluation

## 📊 Model Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

## 📈 Key Findings

* Vehicle age significantly affects resale value.
* Present price is one of the strongest predictors of selling price.
* Fuel type and transmission influence market value.
* Machine learning models can effectively estimate used car prices.

## ✅ Conclusion

The project demonstrates how machine learning can be applied to predict car prices accurately using historical vehicle data. Such models can assist buyers, sellers, and dealerships in making informed pricing decisions.
